Acc Bank API

URL Base

As requisições são feitas em uma URL base sob protocolo HTTPS.

URL: https://api.accbankbrasil.com.br/

Content-Type

O retorno de dados da API será via JSON.

Content-Type: application/json

Swagger

A API possui uma interface Swagger, onde é possível explorar todos os endpoints disponíveis e testar as requisições diretamente.

O Swagger pode ser acessado pelo seguinte link:

https://api.accbankbrasil.com.br/swagger-ui/index.html#/

Endpoints - Agência

GET /agencias

Descrição: Retorna uma lista de todas as agências cadastradas.

Exemplo de resposta:
[
    {
        "id": 1,
        "nome": "Agência Central",
        "endereco": "Rua Principal, 123"
    },
    {
        "id": 2,
        "nome": "Agência Norte",
        "endereco": "Avenida Norte, 456"
    }
]

Buscar Agência por ID

Endpoint: GET /agencias/{id}

Descrição: Retorna os detalhes de uma agência específica com base no ID fornecido.

Exemplo de resposta:
{
    "id": 1,
    "nome": "Agência Central",
    "endereco": "Rua Principal, 123"
}

Atualizar Agência

Endpoint: PUT /agencias/{id}

Descrição: Atualiza as informações de uma agência existente com base no ID fornecido.

Exemplo de corpo da requisição:
{
    "nome": "Agência Central Atualizada",
    "endereco": "Rua Nova, 456"
}

Adicionar Nova Agência

Endpoint: POST /agencias

Descrição: Adiciona uma nova agência ao sistema.

Exemplo de corpo da requisição:
{
    "nome": "Agência Sul",
    "endereco": "Avenida Sul, 789"
}

Remover Agência

Endpoint: DELETE /agencias/{id}

Descrição: Remove uma agência existente com base no ID fornecido.

Endpoints de Cliente

Listar Clientes

Endpoint: GET /clientes

Retorna uma lista de todos os clientes.

Buscar Cliente por ID

Endpoint: GET /clientes/{id}

Retorna os dados de um cliente específico com base no ID.

Atualizar Cliente

Endpoint: PUT /clientes/{id}

Atualiza os dados de um cliente específico.

Adicionar Cliente

Endpoint: POST /clientes

Adiciona um novo cliente.

Remover Cliente

Endpoint: DELETE /clientes/{id}

Remove um cliente com base no ID.

Endpoints de Conta Corrente

Buscar Conta Corrente por ID

Endpoint: GET /contas-correntes/{id}

Retorna os dados de uma conta corrente específica com base no ID.

Listar Contas Correntes

Endpoint: GET /contas-correntes

Retorna uma lista de todas as contas correntes.

Sacar

Endpoint: POST /contas-correntes/{id}/sacar

Realiza um saque em uma conta corrente específica.

Depositar

Endpoint: POST /contas-correntes/{id}/depositar

Realiza um depósito em uma conta corrente específica.

Transferir

Endpoint: POST /contas-correntes/{idOrigem}/transferir

Realiza uma transferência entre contas correntes.

Endpoints de Transação

Extrato Geral

Endpoint: GET /contas-correntes/{idConta}/extrato

Retorna o extrato geral de uma conta corrente específica.

Extrato Mensal

Endpoint: GET /contas-correntes/{idConta}/extrato-mensal?mesAno=MM/yyyy

Retorna o extrato mensal de uma conta corrente específica.

Extrato Anual

Endpoint: GET /contas-correntes/{idConta}/extrato-anual?ano=yyyy

Retorna o extrato anual de uma conta corrente específica.

Extrato Filtrado

Endpoint: GET /contas-correntes/{idConta}/extrato-filtrado?dataInicio=dd/MM/yyyy HH:mm&dataFim=dd/MM/yyyy HH:mm

Retorna o extrato filtrado por período de uma conta corrente específica.
